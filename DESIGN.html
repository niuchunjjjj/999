空状态：出牌区无牌时显示"请选 1–5 张"中性提示
胜利：金边黑底 Toast，居中遮罩，按任意键继续
失败：出牌次数耗尽时自动触发，提示"重新开始"

给开发的话

  • CSS 变量建议：--color-table / --color-accent / --color-card-red / --color-gold，统一在 :root 定义
  • 卡牌动效：transform: translateY(-12px) + box-shadow: 0 0 0 2.5px var(--color-accent)
  • 过渡统一：transition: transform 200ms ease, box-shadow 200ms ease
  • 字体方案：数字/代码用 JetBrains Mono，正文用 Noto Sans SC，fallback 均为 system-ui
  • HUD 数字防抖：font-variant-numeric: tabular-nums，防止数字变化时布局跳动
  • 移动适配：游戏容器固定 1080×640，移动端用 transform: scale() 整体缩放
